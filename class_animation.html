This script does a lot of things and should probably be refactored into multiple classes. a) Creates & Updates Keyframes for position, size and shape of AOI Does this by listening to DraggableAroundCameras and Resizables events b) Calculates position, size and shape for Frames between Keyframes using interpolation c) Animates object based on current frame of video by setting position, size and shape according to calculated frames. Detailed Description

This script does a lot of things and should probably be refactored into multiple classes. a) Creates & Updates Keyframes for position, size and shape of AOI Does this by listening to DraggableAroundCameras and Resizables events b) Calculates position, size and shape for Frames between Keyframes using interpolation c) Animates object based on current frame of video by setting position, size and shape according to calculated frames.
