Detailed Description

Derived from SRanipal example script. This script is responsible for getting the eyedata from the VR-Headset, storing the data during recording as well as carrying out the raycast to check whether or not the gaze is hitting a AOI. Note: Callback runs on a separate thread to report at ~120hz. Unity is not threadsafe and cannot call any UnityEngine api from within callback thread.
