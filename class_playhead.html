virtual void OnPointerExit (PointerEventData eventData)
 If the cursor leaves the area of the clickable item but has previously beed clicked and the mouse is still pressed down, the item makes sure that the timeline still doesn't register as being hovered, because the user is still interacting with the item itself, since the mouse is still pressed down (the user might e.g. be trying to drag an item across the timeline.).
